## 1. 명세를 참고해 타입을 생성하라

- 잘 설계된 API, 명세를 자동으로 생성하는 것은 사용자가 실수를 줄일 수 있게 도와줍니다.
- 사용자에게 명확한 오류 메시지를 제공합니다.
- 사용가능한 모든 값에 대해서 작동한다는 확신을 제공합니다.

## 2. 정리

- 타입 안정성을 위해 명세를 이용해 타입을 생성하라.
- 예외적인 경우들이 문제가 될 수 있으므로 제공되는 명세를 이용해라.
